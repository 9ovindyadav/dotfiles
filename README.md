# dotfiles

### Prerequisites
1. GNU Stow
2. Git

```bash
sudo apt install git stow xclip fzf
```

## Installation
1. Clone the `dotfiles` repo in the `$HOME` directory
```bash
git clone 
cd dotfiles
```
2. Run `stow .` to simlink all the files
```bash
stow .
```

## Note
- `dotfiles` file structure should be same as it is in `$HOME` directory
